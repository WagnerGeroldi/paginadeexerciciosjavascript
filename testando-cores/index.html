Instruções

Uma forma muito bacana de capturar textos em inputs é a propriedade .keyUp

Exemplo:
$('input').keyup(
function () {
const cor = $(this).val()
$('.card-2').css('background', cor).css('transition', '0.9s')
})
 Assim conseguimos manipular algumas funções de forma interessante!
